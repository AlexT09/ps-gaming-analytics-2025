# PS Gaming Analytics 2025

Repositorio de ejemplo para la entrega del proyecto de Visualización de Datos.

## Contenido

- `dash_app/`: Dashboard en Python con Dash y Plotly
- `shiny_app/`: App interactiva en R con Shiny
- `INSTRUCCIONES.md`: Guía para ejecutar el proyecto

## Cómo usar este repositorio

1. Abrir `dash_app/` para ver el dashboard Python.
2. Abrir `shiny_app/` para ver la app Shiny.
3. Ejecutar las instrucciones de `INSTRUCCIONES.md`.
